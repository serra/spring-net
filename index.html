spring-net by serra

My fork of Spring Framework for .NET

Documentation

While learning the framework I find myself (re-)reading the excellent spring docs a lot. I figured I might do some small fixes on the documentation I found while doing this. The results are published here in HTML and pdf format.

I'm working on a good way to do an automatic publish of the docs on my forked branches: github.com/serra/spring-net/issues/4.

I'll put some notes on building and "deploying" the documentation on the /serra/spring-net/wiki.
